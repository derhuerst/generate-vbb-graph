## Installing

```shell
npm install -g generate-vbb-graph
```


## Usage

```
Usage:
    generate-vbb-graph [-p subway,tram]
Options:
    --products  -p  A list of products. These are available:
                    suburban, subway, regional, tram, ferry, bus
Examples:
    generate-vbb-graph -p subway,tram
```
